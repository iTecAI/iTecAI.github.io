Xorien's Lair
Xorien's Lair (or more specifically Xorien's Lair 3) is the third iteration of a centralized D&D 5e compendium, character manager, and collaborative map viewer. This iteration is functionally the final prototype, and is not yet fully scalable. It uses a Python REST API backend, and HTML/CSS/JS for the frontend. This is by far my largest project, taking almost a year to get to this point.

p2p-Local
p2p-Local is a small but versatile Python library for a full-mesh peer-to-peer communication between devices on a local network. It supports multiple channels, and implements UDP broadcasting for neighborhood discovery. Once a link between devices is formed, nodes can call functions on other nodes via the XMLRPC protocol.

GISS
GISS, or the Google Infinite Storage Solution, is a small project that encodes binary (or any other type) files into base64, then stores them in Google Docs. It creates extra files as needed, and can split larger files in between multiple Docs files. This theoretically allows far more binary storage than Google Drive would normally allow, due to the unlimited Docs storage. It was mostly just an exercise in dealing with the Drive APIs, but it is fully functional.

Roh 3.0
Roh 3.0 is the third iteration of a slightly frivolous chatbot based on a fairly simple keyword-detection algortihm. However, the keywords and associated responses are automatically generated based on its own conversations, meaning it does learn very gradually over time. It is currently inhabiting a Discord bot, but is a self-contained library and can be used in many scenarios.
